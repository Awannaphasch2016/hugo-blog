A Note On Straight's Recipe.

Straight’s recipe is similar but not identify to one used by MELPA. 1 A recipe describes which local repository to link the files from and how to clone that repository, if repo is not found locally. With this definition of recipe, one can think of recipe as a build step (recipe) of a package. Ultimately, a package is defined is a collection of files required for package’s build. striaght.el views a package in this manner.
